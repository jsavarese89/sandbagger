Sandbagger Debugging Mode

This page will help diagnose and fix React hooks errors in your application.

Current Issue: React Error #310

This error occurs when the Rules of Hooks are violated in your React components. Specifically, it's happening when:

Hooks cannot be called conditionally or after early returns. Hooks must be called in the exact same order on every render.
Common causes:
  • Calling hooks inside conditional statements (if/else blocks)
  • Calling hooks inside loops
  • Using hooks after an early return statement
  • Missing hook imports (like useCallback)
  • Inconsistent hook ordering between renders
